HTML

Hypertext Markup Language - it is a markup language that is used for defining the building blocks of a webpage. The landing page file should be always named as index.html and the first line of html file should be <!DOCTYPE html >

HTML is not a case senstive language except for DOCTYPE which is usually always used in uppercase letters.
HTML contains elements most of which contains opening and closing tabs like for head and body elements.
There are elements that does not have closing tags for example img. The elements that can have child elements within itslef will have closing tag and since the image element is not allowed to have any child elements it doesnot end with closing tag like </img>


HTML sections

There are two main sections of HTML document - Head and body.
Head section is used to give the browser information about the web page like what is the title of the webpage, styling details, etc.
Body section contains the information that is displayed on the webpage.
This contains the paragraphs, headings and other details.
There are 6 heading types in HTML and it is to be noted that each webpage should contain only one H1 heading and the headings represent the hierarchy of the document, they should not be used just to leverage their font size since each heading tag can be customized using CSS.

Hyperlinks

Link and hyperlink are two terms that are often use interchangebly but they are different. By definition links are just an address or URL location of the target page, and hyperlinks are the elements the user can click on the page to navigate to that target webpage.
Hyperlinks are links to other pages or websites provided within this webpage. Usually they can be created to other pages, external webpages, images, etc. To create these hyperlinks on the html page we use anchor element which is represented as <a href="">
Here href stands for Hypertext reference and contains the URL (Uniform Resource Locator) for the other websites.
Href can contain two types of paths - relative link and absolute links. Relative link paths can be used when we want to navigate to a webpage within our local directories or system. For example, I have created another page to discuss CSS in detail which can be accessed by this link CSS. Here the href conatins value as follows - href = "/css.html".
However, if we want to jump to external websites then the link should start with protocol like https or http. For example, if i want to open new google window, i have to specify the link as follows - href = "https://google.com". And to open the link in a new tab we have to sepcify another attribute called target to "_blank". Google Page
